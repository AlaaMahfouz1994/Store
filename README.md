
To run the project 
  - download the project 
  - Run `npm install` to install the node_modules
  - Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`.

- To login as an Admin, use the follwing credentials: { username: 'admin', password: 'Password0101'}
- To login as a user, use the follwing credentials: { username: 'user', password: 'Password0101'}
